## 0. Why this track exists

The Collatz framework already kills *bounded gliders* (Lock 3 / death
shell / capacity edge) and has massive certificates for cycles and
finite scans. The open boss is **composition of corridor exits into a
global deficit bound** (sub-threshold / unbounded-escape; see
`INVESTIGATION_SUB_THRESHOLD_LEMMA.md`).

G1 is a **feet-wet computational campaign** with two independent
instruments that can falsify or strengthen the "shell-as-composition"
story without attempting a full proof:

| Experiment | Question | Relation to program |
|---|---|---|
| **E1 Letter surgery** | Does the 358-letter support‚Üídrop flip control \(D(m)\) the way W6D-M claims? | Mechanism test for F5 route (358 vs 359) without \(m=359\) search |
| **E2 Exit toll census** | Do upward corridor exits land in the death-shell skin of the wider corridor? | Composition hypothesis for escape |

A third stretch item (**E3 őĒM staircase**) is registered only if E1/E2
complete with wall-clock to spare; it is not load-bearing for G1.

## 3. E1 ‚ÄĒ Letter-flip surgery (hybrid credit words)

### 3.1 Object

Define three credit words on step index \(k \ge 0\):

- **True:** \(c^{\mathrm{true}}_k = \lfloor(k+1)\alpha\rfloor - \lfloor k\alpha\rfloor\), \(\alpha=\log_2 3\), exact integer arithmetic
- **Periodic 22/53:** \(c^{22/53}_k\) = mechanical word of the convergent (per W6D ground-truth convention already used in-repo ‚ÄĒ ledger the exact formula copied from the source file, do not invent a new tiling)
- **Hybrid splice at N:** for chosen splice index \(N\),

\[
c^{(N)}_k =
\begin{cases}
c^{\mathrm{true}}_k & k < N \\
c^{22/53}_k & k \ge N
\end{cases}
\]

Also compute the **reverse hybrid** \(c^{(N,\mathrm{rev})}\) (periodic first, true after) as a control family.

### 3.2 Measurement

For each word family and each \(N\) in the grid below, compute \(D(m)\)
using the same dense shell / minimax instrument validated by V1‚ÄďV2,
for \(m \in \{2,\ldots,m_{\max}\}\) with \(m_{\max}\) at least 12 and as
high as dense enumeration allows under a documented state/RSS cap
(ledger the cap before the run).

**N grid (frozen):**

```
N ‚ąą {0, 53, 106, 200, 300, 350, 357, 358, 359, 360, 400, 530}
```

- \(N=0\): pure periodic (all letters from 22/53)
- \(N=\infty\) equivalent: pure true word (include as explicit row `N=inf`)
- Critical cluster around 358 is intentional (first true-vs-periodic divergence scale)

### 3.3 Deliverables

- Table: `artifacts/e1_hybrid_D.csv` with columns
  `family, N, m, D, support_count_window, notes`
- Window definition **end-anchored** at the step count the D-instrument
  actually uses (must match shell_probe / underlock convention;
  ledger the exact window). Also record start-anchored support counts
  as a control column, not as the primary law.
- Script(s) under `scripts/`; logs under `artifacts/`.

### 3.5 Registered predictions (Grok, frozen 2026-07-08, pre-data)

**P1.1 (support-cost / W6D-M style ‚ÄĒ primary):**  
For m in the dense range, \(D_{\mathrm{true}}(m)\) and \(D_{22/53}(m)\)
agree until the constrained window can "see" letter 358. Hybrids with
\(N \le 357\) that still place letter 358 in the periodic regime match
periodic D; hybrids that place the true letter 358 inside the window
match true D. **Confidence: 0.55** (mechanism is attested but phase
pinning has burned this program before).

**P1.2 (no small-m leak):**  
For all m‚Č§12, true and 22/53 D agree **exactly** (both laws live in the
C‚Č§147 agreement zone). Hybrid N-variation produces **zero** D changes
at m‚Č§12. **Confidence: 0.85**.  
If P1.2 fails, either the instrument is wrong or small-m already feels
the splice ‚ÄĒ both are major.

**P1.3 (first visible split scale):**  
The first m where true vs periodic D diverge under this instrument is
**not** visible below the dense cap we can run; E1's job at small m is
mechanism-interpolation shape, not deciding F5. **Confidence: 0.80**.

**P1.4 (F5 implication ‚ÄĒ conditional, not a direct measurement):**  
If P1.1 holds in the form "one support letter in the end-anchored window
moves D by exactly 1," then at m=359 the true word's one fewer support
vs periodic (149 vs 150 on the start-anchored census in SYNTHESIS)
favors **D(359)=148 ‚áí edge 359**. This is a *conditional route
reinforcement*, not an F5 decision. F5 remains OPEN unless a later
track closes it under frozen algebraic gates. **Confidence on the
conditional arrow: 0.45** (explicitly low; fitting-family trap).

**P1.5 (reverse hybrid asymmetry):**  
Reverse hybrids (periodic prefix, true suffix) differ from forward
hybrids at the same N ‚ÄĒ end-anchored windows care about the *suffix*.
**Confidence: 0.70**.

**Refutation language (binding):**

- If P1.1 fails (D ignores the splice that changes window support

## 4. E2 ‚ÄĒ Exit toll census (upward breaches vs shell skin)

### 4.1 Object

**Hypothesis (composition / shell toll):**  
When a concrete orbit (or CRT-reconstructed breach witness) exits
corridor width \(C\) upward into \(C+1\) (or higher), the state at the
exit step ‚ÄĒ recorded as ceiling-distance \(\delta = C - d\) in the
*pre-exit* corridor, and as \(\delta' = (C+1) - d'\) post-exit ‚ÄĒ
lies in the **marginal shell layer** of the wider corridor more often
than a null model predicts.

Null model (frozen): uniform over live nonzero-class residues at the
same \((C, m_{\mathrm{obs}}, d)\), or if full live sets are too large,
uniform over \(\delta \in \{0,\ldots,C\}\) with residue class ‚Č†0 mod 3.
Ledger which null is used; prefer the live-set null when feasible.

### 4.2 Data sources (existing certificates first)

Priority order:

1. Published breach-follow / last-survivor reconstructions cited in
   `COLLATZ_PROOF.md` / Lock3 results (paths must be located and
   listed in the ledger before analysis)
2. `data/runs/` directories matching `*breach*`, `*last*survivor*`,
   high-reserve orbits (`80049391`, `319804831`, ‚Ä¶)
3. If fewer than **30** clean upward-exit events are recoverable from
   existing artifacts, generate a **bounded** fresh sample:
   - odd starts from the known high-reserve list + 100 random odds in
     \([1, 10^7)\)
   - record every upcrossing of a running max corridor
   - step cap and start list frozen in the ledger *before* the run

### 4.3 Measurement per exit event

For each upward exit event record:

```
start_n, step_k, C_before, C_after, d_before, d_after,
a_k, c_k, residue_mod_3^m, m_used, delta_before, delta_after,
in_shell_before, in_shell_after, shell_depth_D_of_m
```

`m_used`: start at m=6 (cheap) and m=8 if affordable; do not claim
shell membership at m that the one-heartbeat proxy cannot see for the
relevant depth (respect F9: one-heartbeat liveness is m-blind for
m‚Č≥54 ‚ÄĒ E2 uses short forward orbits and modest m only).

`in_shell_*`: membership in the F7 death shell at ceiling-distance
keying ‚ÄĒ reuse `shell_probe` / dense shell tables where possible.

- `artifacts/e2_exit_events.csv`
- `artifacts/e2_summary.json` (counts, rates, null comparison, CIs)
- Scripts under `scripts/`

### 4.6 Registered predictions (Grok, frozen 2026-07-08, pre-data)

**P2.1 (shell toll ‚ÄĒ primary):**  
Post-exit states are enriched in the top shell layers
(\(\delta' \le D(m)\) band / dead-or-marginal) relative to the null.
Enrichment factor ‚Č• **1.5√ó** on the primary sample. **Confidence: 0.50**.

**P2.2 (forced a=1 at drop-ceiling exits):**  
A large fraction of pure ceiling upcrossings occur at drop phases with
\(a=1\) (as COLLATZ_PROOF already argues). Rate of \(a=1\) among
upcrossings ‚Č• **0.70**. **Confidence: 0.75**.

**P2.3 (no free climb on live interior):**  
Exits from deep interior (\(\delta \ge 3\) below ceiling at modest C)
are rare (<20% of exits). Most exits are ceiling-adjacent.
**Confidence: 0.65**.

**P2.4 (witness fate):**  
Among CRT / high-reserve witnesses that upcross, subsequent return
below the pre-exit corridor within 200 odd-steps remains the dominant
fate (‚Č•80%), consistent with prior 50/50 and 30/30 collapse reports ‚ÄĒ
E2 re-measures on its own sample. **Confidence: 0.70**.

**Refutation language (binding):**
- If P2.1 fails (no enrichment, or enrichment <1.1√ó with tight CI) ‚Üí
  shell-toll composition hypothesis **REFUTED** at this operationalization;
  escape composition needs a different object (dual-game / realizability).
- If P2.2 fails badly (a=1 rate <0.4) ‚Üí either sample is not ceiling
  exits or the proof text's exit mechanism needs revision.
- E2-G1 fail ‚Üí **INCONCLUSIVE**, not refutation.
